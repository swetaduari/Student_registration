# 👩🏻‍🎓Student Registration System (Dynamic Web Project)

## 📂Project Description
The **Student Registration System** is a web-based application developed to manage student records efficiently. The system allows an administrator to securely log in and perform operations such as adding, viewing, updating, and deleting student information from the database.

The application includes an **Admin Sign-in page** designed using **HTML, CSS, and JavaScript**, with JavaScript used to create animations and enhance the user interface for a better user experience.

The backend of the system is built using **Java Servlets**, which handle user requests and implement the core business logic of the application. The system supports full **CRUD operations (Create, Read, Update, Delete)** for managing student data.

For database connectivity, the project uses **JDBC (Java Database Connectivity)** to establish communication between the Java application and the database. Student data is securely stored and retrieved from the database through JDBC.

The application runs on the **XAMPP server**, providing a suitable environment for deployment and testing. Additionally, **session management** is implemented to maintain secure admin login sessions and prevent unauthorized access to the system.

Overall, this project demonstrates the integration of **frontend web technologies with Java-based backend development and database connectivity**, providing an efficient solution for managing student registration information.
